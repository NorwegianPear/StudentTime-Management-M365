Hei Adnan,

Takk for tilbakemeldingen — flott at ledelsen er positiv! Her er en fullstendig kostnadsoversikt som kan brukes direkte i budsjettet, med priser per måned, per kvartal og per år.

Viktig å vite: Siden Azure-abonnementet allerede er opprettet via Atea, vil BISS faktureres direkte av Atea som CSP-partner (Cloud Solution Provider) — ikke av Microsoft. Atea konsoliderer Azure-kostnadene og utsteder faktura til BISS på norsk, i NOK, med norsk MVA.

🏢 Hvordan fakturering via Atea (CSP) fungerer

Faktureringsmodell Tilgjengelig via Atea? Merknad
📅 Månedlig ✅ Ja — standard Atea fakturerer BISS månedlig basert på faktisk forbruk. Ingen binding. Mest fleksibelt.
📅 Kvartalvis ⚠️ Kan avtales med Atea Azure har ikke innebygd kvartalsfakturering — men Atea kan samle opp 3 måneders forbruk og sende én faktura per kvartal som en tilpasset faktureringsavtale. Dette avtales direkte med Atea-kundeansvarlig. Kostnaden er uendret.
📅 Halvårlig ⚠️ Kan avtales med Atea Samme som kvartalvis — kan konsolideres av Atea på forespørsel.
📅 Årlig ✅ Ja — via Reserved Instances For portal-alternativet kan App Service B1 reserveres for 1 år med fast månedspris — gir ~35% rabatt. Atea fakturerer fortsatt månedlig, men til den lavere reserverte prisen. Alternativt kan full årssum betales på én faktura (se tabell nedenfor).
💡 For budsjettformål: Prisene nedenfor er eks. MVA. Azure-tjenester faktureres uten MVA til norske bedriftskunder (omvendt avgiftsplikt / snudd avregning). Sjekk med BISS sin regnskapsavdeling for korrekt MVA-behandling.

🧾 Faktureringsflyt — Atea som CSP-partner

Slik fungerer det når BISS betaler via Atea CSP:
  1. Microsoft → Atea: Microsoft fakturerer Atea som CSP-partner månedlig for BISSs faktiske Azure-forbruk (til CSP-engrosrabatt).
  2. Atea → BISS: Atea utsteder faktura til BISS i NOK, med norsk fakturanummer, norsk adresse og egne betalingsbetingelser. Fakturaen inkluderer Azure-forbruk + eventuell Atea CSP-serviceandel.
  3. Betalingsbetingelser: Standard 30 dager netto for norske kunder, men dette kan tilpasses (f.eks. 15 dager, 45 dager, kvartalsfaktura).
  4. MVA: Azure-tjenester fra Microsoft faktureres vanligvis uten MVA til norske bedriftskunder (snudd avregning §15 mval). Atea vil håndtere dette korrekt på fakturaene.
  5. Kvartalsfakturering: Ønsker BISS én faktura per kvartal i stedet for månedlig, avtales dette direkte med Atea kundeansvarlig — Atea akkumulerer da tre måneders forbruk til én faktura.
